# Chrome Web Store 商店列表

## 扩展名称
Use AI Writer - AI写作助手

## 简短描述（132字符以内）
在任何网页上使用AI生成高质量内容。支持博客、邮件、社交媒体等多种格式，一键生成、复制、插入。

## 详细描述
Use AI Writer Chrome扩展让您在任何网页上都能使用强大的AI写作助手。无论是撰写博客、回复邮件还是发布社交媒体，都能快速生成高质量内容。

### 🎯 核心功能

**多场景支持**
- 📝 博客文章：生成长篇内容，结构清晰
- 📧 电子邮件：专业邮件模板，礼貌得体
- 💬 社交媒体：简短有力，吸引眼球
- ⚙️ 自定义格式：灵活适配各种需求

**智能文本处理**
- ✨ 改进文本：优化语法、提升表达
- 📋 总结内容：提取关键信息，简明扼要
- 📖 扩展内容：增加细节，丰富表达

**便捷操作**
- 🖱️ 右键菜单：选中文本后快速操作
- 🎨 浮动按钮：随时呼出AI助手
- 📋 一键复制：快速复制生成内容
- 📥 智能插入：直接插入到页面输入框

**历史记录**
- 🕐 自动保存：记录最近50条生成内容
- 🔍 快速查看：随时回顾历史生成
- 🗑️ 一键清空：保护隐私，随时删除

### 🔒 隐私保护

- ✅ 数据本地存储：历史记录仅保存在您的浏览器

- ✅ 无追踪：不收集浏览历史或个人信息

### 💡 使用方法

1. **安装扩展**：从Chrome Web Store安装
2. **登录账号**：点击扩展图标，使用Use AI Writer账号登录
3. **开始使用**：
   - 点击页面右下角的浮动按钮
   - 或选中文本后使用右键菜单
   - 或选中文本后自动弹出操作菜单
