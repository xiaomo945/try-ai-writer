# Design Standards
- Primary: emerald-600 (#059669), Accent: teal-500 (#14b8a6)
- Fonts: Playfair Display (headings, 800 weight) + Inter (body, 400 weight)
- Spacing: 4px grid system, container max-w-7xl
- Cards: rounded-2xl (16px), shadow-sm to hover:shadow-xl shadow-emerald-500/5
- Touch targets: min 44x44px, focus-visible outline on all interactive elements
- Dark mode: global support via class strategy
- Animations: transition-all duration-300 ease-out
