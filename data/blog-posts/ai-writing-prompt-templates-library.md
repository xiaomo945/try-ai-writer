Having a library of proven prompt templates saves hours of prompt engineering time and ensures consistent quality across your content. This collection provides forty-five ready-to-use prompts organized by content type. Simply fill in the bracketed placeholders and start generating.

## How to Use These Templates

Each template contains bracketed placeholders like [TOPIC] or [AUDIENCE] that you replace with your specific details. For the principles behind effective prompts, read our [Prompt Engineering Master Guide](/blog/ai-writing-prompt-engineering-master-guide). Customize freely—add constraints, adjust tone, or modify structure to match your needs.

## Blog Article Prompts

### 1. Comprehensive Guide

```
Write a comprehensive guide about [TOPIC] for [AUDIENCE]. Context: [BRIEF BACKGROUND]
The guide should: be [WORD COUNT] words, use [TONE] tone, start with a hook addressing [PAIN POINT], cover [KEY POINTS], include practical examples, end with actionable next steps. Do not use jargon without explanation.
```

### 2. Listicle

```
Write a listicle titled "[TITLE]" featuring [NUMBER] items about [TOPIC].
Each item: bold subheading, 2-3 paragraphs, one example, practical tip. Audience: [AUDIENCE] | Tone: [TONE] | Keyword: [KEYWORD]
```

### 3. How-To Tutorial

```
Write a step-by-step tutorial teaching [AUDIENCE] how to [TASK]. Difficulty: [BEGINNER/INTERMEDIATE/ADVANCED]
Structure: Introduction, numbered steps (what to do, why it matters, common mistakes), troubleshooting, summary with next steps. Tone: Encouraging | Length: [WORD COUNT] words
```

### 4. Comparison Article

```
Write a comparison between [OPTION A] and [OPTION B] for [AUDIENCE].
Cover: [DIMENSION 1], [DIMENSION 2], [DIMENSION 3], [DIMENSION 4], [DIMENSION 5].
For each dimension, explain both options and declare a winner. Include summary table and scenario-based recommendations.
```

### 5. Thought Leadership

```
Write a thought leadership article arguing [POSITION] for [AUDIENCE].
Support with: [EVIDENCE TYPE 1], [EVIDENCE TYPE 2], [EVIDENCE TYPE 3]. Address the strongest counterargument. Use specific examples. Tone: Confident but not arrogant | Length: [WORD COUNT] words
```

## Email Marketing Prompts

### 6. Welcome Email

```
Write a welcome email for new subscribers to [BRAND/PRODUCT]. Voice: [VOICE] | Product: [DESCRIPTION]
Include: warm greeting, what to expect from [FREQUENCY] emails, one immediate value, soft product intro, CTA. Length: 150-250 words
```

### 7. Promotional Email

```
Write a promotional email for [PRODUCT/SERVICE] targeting [AUDIENCE].
Offer: [OFFER] | Urgency: [DEADLINE] | Key benefit: [BENEFIT]
Include: 3 subject line options (under 50 chars), desire-driven opening, 3 benefit bullets, social proof, CTA with urgency.
```

### 8. Newsletter

```
Write a weekly newsletter for [AUDIENCE] about [TOPIC].
Sections: Opening thought (2-3 sentences), main insight, resource recommendation, quick tip. Tone: [TONE] | Length: 300-400 words
```

### 9. Re-Engagement Email

```
Write a re-engagement email for subscribers inactive for [TIME PERIOD]. Brand: [BRAND] | Value: [DIFFERENTIATOR]
Include: acknowledge absence without guilt, remind of value, offer something specific, option to update preferences. Tone: Warm
```

### 10. Transactional Email

```
Write a [confirmation/shipping/receipt] email for [PRODUCT/SERVICE].
Include: clear subject line, confirmation details, [SPECIFIC DETAILS], next steps, optional related suggestion. Tone: Professional | Length: Under 200 words
```

## Social Media Prompts

### 11. LinkedIn Post

```
Write a LinkedIn post about [TOPIC] for [AUDIENCE].
Format: Hook (under 15 words), 3-5 expanding sentences, actionable takeaway, engagement question. Tone: Professional but conversational
```

### 12. Twitter/X Thread

```
Write a [NUMBER]-tweet thread about [TOPIC].
Tweet 1: Hook with bold claim. Tweets 2-[N-1]: One key point per tweet. Final: Summary and CTA. Style: Punchy, direct.
```

### 13. Instagram Caption

```
Write an Instagram caption for [TOPIC]. Include: attention first line, 2-3 sentences of value, CTA, 5-8 hashtags. Tone: [TONE] | Length: Under 150 words
```

### 14. TikTok Script

```
Write a [LENGTH]-second TikTok script about [TOPIC].
Structure: Hook (first 3 seconds), setup, value delivery, CTA (last 3 seconds). Style: Fast-paced, conversational.
```

### 15. YouTube Description

```
Write a YouTube description for a video about [TOPIC]. Include: compelling first 2 lines, brief summary, timestamps for [SECTIONS], links, subscribe CTA, 5-8 keywords.
```

## Landing Page and Marketing Prompts

### 16. SaaS Landing Page

```
Write SaaS landing page copy for [PRODUCT]. Customer: [AUDIENCE] | Differentiator: [UNIQUE VALUE]
Sections: Hero (headline, subheadline, CTA), Problem (3 pain points), Solution, Features (4-6 benefit-focused), Social proof, Final CTA.
```

### 17. Lead Magnet Page

```
Write a lead magnet page for [RESOURCE]. Audience: [AUDIENCE] | Frustration: [PAIN POINT] | Outcome: [RESULT]
Include: outcome-driven headline, value subheadline, 5 content bullets, 3 credibility indicators, form CTA.
```

### 18. Case Study

```
Write a case study about [CLIENT] using [PRODUCT]. Background: [SITUATION] | Challenge: [PROBLEM] | Solution: [HOW USED] | Results: [METRICS]
Structure: Title with result, summary, challenge, solution, results with numbers, quote placeholder, CTA.
```

### 19. White Paper

```
Write a white paper about [TOPIC] for [AUDIENCE]. Thesis: [ARGUMENT] | Evidence: [DATA POINTS]
Structure: Executive summary, problem landscape, analysis ([3-4 FINDINGS]), implications, 3-5 recommendations, conclusion.
```

### 20. Press Release

```
Write a press release about [ANNOUNCEMENT] for [COMPANY]. What: [ANNOUNCEMENT] | Why: [SIGNIFICANCE] | When: [TIMELINE] | Who: [KEY PEOPLE]
Include: headline, dateline, lead paragraph, 2-3 detail paragraphs, quote, boilerplate, contact placeholder.
```

## SEO and Product Prompts

### 21. Pillar Page

```
Write a pillar page about [BROAD TOPIC] for [AUDIENCE]. Subtopics: [SUBTOPIC 1-5]. For each: H2 with keyword variation, 200-300 words, internal link placeholder. Primary keyword: [KEYWORD] | Length: 2,500-3,500 words
```

### 22. Meta Description

```
Write 3 meta descriptions for a page about [TOPIC]. Under 155 characters, include keyword [KEYWORD], include CTA. Differentiate: benefit-focused, question-based, action-oriented.
```

### 23. Product Description

```
Write a product description for [PRODUCT NAME]. Features: [LIST] | Benefits: [TRANSLATE EACH] | Buyer: [PERSONA]
Include: attention-grabbing opening, benefit-led features, social proof placeholder, CTA. Tone: [TONE]
```

### 24. Amazon Listing

```
Write an Amazon listing for [PRODUCT]. Differentiators: [UNIQUE]
Include: title (under 200 chars, keyword-rich), 5 bullet points (feature + benefit, under 200 chars each), description (under 1,000 words). Keywords: [KEYWORDS]
```

## Editing and Refinement Prompts

### 25. Content Rewrite

```
Rewrite the following to be more [concise/engaging/persuasive/professional]. [PASTE CONTENT]
Maintain the core message but: [IMPROVEMENT 1], [IMPROVEMENT 2], [IMPROVEMENT 3]. Target length: [WORD COUNT] words
```

### 26. Tone Adjustment

```
Adjust the tone from [CURRENT TONE] to [DESIRED TONE]. [PASTE CONTENT]
Key changes: [CHANGE 1], [CHANGE 2], [CHANGE 3]. Keep all facts intact.
```

### 27. Content Expansion

```
Expand this outline into a full article. [PASTE OUTLINE]
For each section: add detailed explanations, specific examples, transitions. Maintain [TONE] tone. Target length: [WORD COUNT] words
```

## Research and Planning Prompts

### 28. Content Brief

```
Create a content brief for an article about [TOPIC].
Include: target keyword and search intent, 3 title options, audience description, key points, word count, internal linking opportunities, competitor gaps.
```

### 29. Topic Cluster Plan

```
Plan a content cluster around [CORE TOPIC].
Provide: 1 pillar page topic, 8-10 supporting topics, keyword suggestions, internal linking structure, priority ranking.
```

33. **Slide Deck**: "Write talking points for a 10-slide presentation about [TOPIC], 2-3 key points per slide."
34. **Podcast Intro**: "Write a 30-second podcast intro for [TOPIC] with [GUEST NAME]."
35. **Webinar Description**: "Write a webinar description for [TOPIC] with title, 3 learning outcomes, and target audience."
36. **Job Description**: "Write a job description for [ROLE] at [COMPANY TYPE] with responsibilities, requirements, and culture."
37. **Onboarding Series**: "Outline a 5-email onboarding sequence for [PRODUCT] with subject lines and key content."
38. **Competitive Analysis**: "Outline a competitive analysis comparing [YOUR PRODUCT] with [COMPETITORS]."
39. **Success Story**: "Write a success story framework with challenge, solution, results, and quote placeholders."
40. **Social Calendar**: "Create a one-week social media calendar for [BRAND] across [PLATFORMS]."

42. **Value Proposition**: "Write a value proposition for [PRODUCT]: We help [AUDIENCE] achieve [OUTCOME] by [METHOD]."
43. **Elevator Pitch**: "Write a 30-second pitch for [PRODUCT/SERVICE] covering problem, solution, and differentiation."
